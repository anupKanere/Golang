# Golang Learning Repository

This repository contains a collection of Go examples, data structures, design patterns, and interview questions.

## Structure

- **basics**: Fundamental Go concepts (variables, loops, structs, interfaces, etc.).
- **data-structures**: Implementations of common data structures (Linked Lists, Hash Tables, etc.).
- **design-patterns**: Examples of Go design patterns.
- **interview-questions**: Solutions to common Go interview questions.
- **practice-questions**: Additional practice problems.

## Prerequisites

- Go 1.24 or higher

## Running Examples

Each subdirectory typically contains a `main.go` file. You can run them using:

```bash
go run path/to/file/main.go
```

For example:

```bash
go run basics/hello-world/main.go
```
